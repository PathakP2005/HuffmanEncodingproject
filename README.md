This project implements Huffman Coding, a lossless data compression algorithm.  
It encodes text into binary form and decodes it back to the original text.

## Features
- Build character frequency map
- Create Huffman Tree
- Generate binary codes
- Encode input text
- Decode encoded text

## Project Structure
src/com/huffmanproject/
- HuffmanNode.java
- HuffmanEncoder.java
- HuffmanDecoder.java
- Main.java

## How to Run
1. Clone the project:
   git clone https://github.com/PathakP2005/HuffmanEncodingproject.git

2. Open in IntelliJ IDEA / Eclipse

3. Run Main.java

## Example
Input:
hello world

Encoded Output:
101010111001...

Decoded Output:
hello world
