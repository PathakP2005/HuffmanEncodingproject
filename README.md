Huffman Encoding & Decoding – Java Implementation

A Java-based implementation of the Huffman Coding algorithm used for efficient lossless text compression.
## Features
- Generates character frequency map
- Constructs Huffman Tree using PriorityQueue
- Produces optimal binary codes
- Encodes and decodes text efficiently

 ## Project Structure
src/
└── com/huffmanproject/
    ├── HuffmanNode.java
    ├── HuffmanEncoder.java
    ├── HuffmanDecoder.java
    └── Main.java
    
## How to Run
1. Clone the repository
   git clone https://github.com/PathakP2005/HuffmanEncodingproject.git

2. Open in IntelliJ IDEA or Eclipse

3. Run the Main.java file

## Example
Input:
hello world

Encoded:
101011001...

Decoded:
hello world
